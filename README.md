# pixelationImage React的图片像素化
```sh
npm install
npm run dev
```
or   
```sh
yarn install
yarn dev
```

#### main中使用图片缩小方法，效果不是很好

#### pixelate-by-rgb 中使用rgb计算，并且可以调整rgb值，最终使用这个组件

tips: 像素度超过12以后就是马赛克了，观感很不好，后续有空再优化
